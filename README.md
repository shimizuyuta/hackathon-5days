## 1. コンセプト・目的・機能

### コンセプト

  + <h3>「色味からコスメを探す」ということで、カラーコードをモチーフにしました</h3>





### 目的
  + ブランドの垣根を超えて、コスメを一覧で見たい！
  + 細かい色味からコスメを探したい！
  
### 機能


  + 類似色のコスメ一覧
  + コスメの詳細を見ることができる
  + カテゴリ別に、色からコスメを検索できる
  + 色味を少しだけ変更して探す
