## 1. コンセプト・目的・使い方

### コンセプト
<h5>色味からコスメを探す」ということで、カラーコードをモチーフにしました！</h5>





### 目的
  + ブランドの垣根を超えて、コスメを一覧で見たい！
  + 細かい色味からコスメを探したい！
  
### 使い方

<h4>カテゴリ別に、色からコスメを検索できる</h4>
<h4>コスメの詳細を見ることができる</h4>
